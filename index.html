Perguntas Frequentes

Tire suas dúvidas sobre o Escola Log

Como funciona o reconhecimento facial antifraude?
Nosso sistema utiliza inteligência artificial avançada para detectar tentativas de fraude, como uso de fotos, vídeos ou máscaras. A tecnologia analisa características biométricas únicas e detecta sinais vitais para garantir que apenas pessoas reais sejam identificadas.
Quais perfis existem no sistema?
O sistema possui perfis específicos para Alunos, Professores, Coordenadores, Diretores e Responsáveis. Cada perfil tem acesso personalizado às funcionalidades relevantes, garantindo segurança e organização dos dados.
Como é feita a integração com WhatsApp?
Integramos diretamente com a API oficial do WhatsApp Business para envio automático de notificações sobre frequência, notas e comunicados importantes. Os responsáveis recebem informações em tempo real.
O sistema funciona offline?
Sim! O sistema possui modo offline que sincroniza automaticamente quando a conexão é restabelecida. Isso garante que a frequência seja registrada mesmo sem internet.
Como é garantida a segurança dos dados?
Seguimos rigorosamente a LGPD, utilizamos criptografia AES-256, autenticação multifator e armazenamento em servidores seguros no Brasil. Todos os dados biométricos são criptografados e nunca compartilhados.
Qual o tempo de implementação?
A implementação completa leva de 1 a 3 semanas, incluindo instalação dos equipamentos, treinamento da equipe e migração dos dados existentes. Oferecemos suporte completo durante todo o processo.
O sistema gera relatórios personalizados?
Sim! Geramos relatórios detalhados de frequência, desempenho acadêmico, estatísticas por turma e período. Os relatórios podem ser exportados em PDF ou Excel e personalizados conforme a necessidade da escola.
Como funciona o suporte técnico?
Oferecemos suporte técnico 24/7 via WhatsApp, telefone e sistema interno. Nossa equipe especializada resolve questões rapidamente e oferece treinamentos contínuos para maximizar o uso da plataforma.
É possível integrar com outros sistemas?
Sim! Nosso sistema possui APIs abertas para integração com sistemas de gestão escolar existentes, plataformas de ensino e outros softwares educacionais que a escola já utiliza.
Qual o investimento necessário?
O investimento varia conforme o número de alunos e funcionalidades desejadas. Oferecemos planos flexíveis e personalizados. Entre em contato para uma proposta específica para sua instituição.

Política de Privacidade e LGPD

1. Informações Gerais

Esta Política de Privacidade descreve como a Escola Log coleta, usa, armazena e protege suas informações pessoais, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

2. Dados Coletados

Coletamos os seguintes tipos de dados:

  • Dados de identificação: nome, CPF, RG, data de nascimento
  • Dados de contato: telefone, e-mail, endereço
  • Dados biométricos: imagens faciais para reconhecimento (com consentimento)
  • Dados de navegação: logs de acesso, cookies, endereço IP
  • Dados acadêmicos: frequência, notas, histórico escolar

3. Finalidade do Tratamento

Utilizamos seus dados para:

  • Controle de frequência e presença
  • Comunicação com responsáveis
  • Gestão acadêmica e administrativa
  • Segurança e monitoramento do ambiente escolar
  • Cumprimento de obrigações legais

4. Base Legal (LGPD)

O tratamento de dados pessoais é realizado com base em:

  • Consentimento do titular ou responsável legal
  • Execução de contrato educacional
  • Cumprimento de obrigação legal
  • Exercício regular de direitos
  • Proteção da vida e segurança física

5. Compartilhamento de Dados

Seus dados podem ser compartilhados com:

  • Autoridades educacionais competentes
  • Prestadores de serviços terceirizados (sob contrato de confidencialidade)
  • Autoridades públicas, quando exigido por lei

6. Seus Direitos (LGPD)

Você tem direito a:

  • Confirmação da existência de tratamento
  • Acesso aos dados
  • Correção de dados incompletos ou inexatos
  • Anonimização, bloqueio ou eliminação
  • Portabilidade dos dados
  • Eliminação dos dados tratados com consentimento
  • Revogação do consentimento

7. Segurança dos Dados

Implementamos medidas técnicas e organizacionais adequadas para proteger seus dados contra acesso não autorizado, alteração, divulgação ou destruição.

8. Retenção de Dados

Mantemos seus dados pelo período necessário para cumprir as finalidades descritas ou conforme exigido por lei.
